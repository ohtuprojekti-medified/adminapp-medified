# adminapp-medified-frontend

## Scripts

### Run
`npm start`

### Test
`npm run test`

### Testcoverage
`npm run test-coverage`

### Lint
`npm run lint`

### Clean code
`npm run lint-fix`

### Open Cypress UI
`npm run cypress:open`

### Run Cypress tests in console
`npm run test:e2e`

### Generate jsdoc documentation
`npm run jsdoc`

### Generate jsdoc documentation and open in default browser
`npm run jsdoc:open`
