# adminapp-medified-backend
Backend for adminapp.

## Scripts
### Run server
npm start
### Development-mode
npm run dev\
Runs server but restarts it when chages are made
### Run tests
npm run test
### Eslint
npm run lint\
Notifies for wrong style in code
### Lint and fix
npm run lint-fix\
Fixes wrong style
### Run server in test environment
npm run start:test
