# adminapp-medified-backend
Adminapp-medfield sovelluksen backend.

## Skriptit
### Ajo
npm start
### Development-tila
npm run dev\
Ajaa backendin normaalisti, mutta tiedostojen muuttuessa, backend käynnistyy uudestaan
### Testien ajo
npm run test
### Lint
npm run lint\
Huomauttaa väärästä tyylistä koodissa
### Lint ja fix
npm run lint-fix\
Siivoaa koodin
